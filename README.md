# FastAPI_test
### Установка 
```
git clone https://github.com/Flowmikro/FastAPI_test.git
```
Перейти в директорию 
```
FastAPI_test
```
#### Через Docker
Прописать `.env.docker`. Есть примеры `.env.docker.example`  
Запустить билд 
```
docker-compose up -d --build  
```
Документация
```
http://localhost:8000/docs/
```
